Detailed Description

The boost.system category for errors returned by the ipc::transport module.

Think of it as the polymorphic counterpart of error::Code, and it kicks in when, for Error_code ec, something like ec.message() is invoked.

Note that this class's declaration is not available outside this translation unit (.cpp file), and its logic is accessed indirectly through standard boost.system machinery (Error_code::name() and Error_code::message()). This class enables those basic features (e.g., err_code.message() returns the string description of the error err_code) to work.

This is the "engine" that allows ec.message() to work, where ec is an Error_code with an error::Code value.
