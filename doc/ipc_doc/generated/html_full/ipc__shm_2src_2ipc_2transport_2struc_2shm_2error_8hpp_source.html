39 /**
40 * Structured message serialization: SHM-session `lend_object()` indicated failure when preparing an encoding

46 /**
47 * Structured message deserialization: SHM-session `borrow_object()` indicated failure when interpreting an encoding

49 * process cannot be reached.
50 */
51 S_DESERIALIZE_FAILED_SESSION_HOSED,

53 /// SENTINEL: Not an error. This Code must never be issued by an error/success-emitting API; I/O use only.
